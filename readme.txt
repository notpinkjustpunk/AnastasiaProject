                                Проект: DLA Snowflake Growth Simulation 
Цель проекта: Создать симуляцию роста снежинки, используя алгоритм диффузионно-ограниченной агрегации (DLA).

DLA- класс, реализующий алгоритм диффузно-ограниченной агрегации.
Процесс построения агрегата начинается с единственной частицы, расположенной в середине прямоугольной сетки.
Случайным образом вычисляется первоначальное положение следующей частицы. Эта частица начинает блуждания по линиям
сетки, при этом на каждом шаге она с вероятностью 1/4 может переместиться на клетку вниз, вверх, вправо или влево.
Блуждание продолжается до тех пор, пока частица не окажется соседней с какой-либо из частиц, входящих в агрегат. 
Тогда она останавливается и закрепляется на своей позиции. Затем происходит бросание новой частицы.

Описание
Проект содержит два основных режима работы:
1. Создание GIF: Программа сохраняет кадры роста снежинки и создает анимацию в формате GIF.
2. Анимация в реальном времени: Программа отображает процесс роста снежинки в реальном времени.
Для работы программы необходимы следующие библиотеки: numpy, matplotlib, seaborn, imajeio.

Использование
Запуск анимации в реальном времени: Для запуска анимации в реальном времени, используется следующий код:
if __name__ == "__main__":
    dla = DLA(size=201)  
    dla.animate(num_particles=1000)
Создание GIF: Для создания GIF анимации, используется следующий код:
if __name__ == "__main__":
    dla = DLA(size=201)  
    dla.run(num_particles=1000)

Функции:
1.DLA.__init__(self, size=101) — Инициализация объекта DLA.
2.DLA.add_particle(self) — Добавляет частицу, которая начинает случайное блуждание от границы.
3.DLA.get_random_edge_position(self) — Возвращает случайную начальную позицию частицы на краю матрицы.
4.DLA.random_walk(self, x, y) — Выполняет случайное перемещение частицы.
5.DLA.has_neighbor(self, x, y) — Проверяет наличие соседних частиц.
6.DLA.capture_frame(self) — Сохраняет текущий статус матрицы для создания GIF.
7.DLA.create_gif(self) — Создает GIF из сохраненных кадров.
8.DLA.run(self, num_particles) — Запускает процесс роста и создания GIF.
9.DLA.animate(self, num_particles) — Запускает анимацию процесса роста в реальном времени.
